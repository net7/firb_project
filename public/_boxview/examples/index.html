BoxView Examples

Please remember that...

  • BoxView and the History Box can re-open only boxes loaded through Ajax, basicly all of the examples where you see some dog.
  • For the same reason, when you try to use an URL of a boxview (either short or long) to open a new page, the boxview will load only boxes that can be loaded through Ajax calls.
  • In all of the examples where AnchorMan is used, your browser's back button will undo your last BoxView interaction: collapse, open, close a box or reorder them.
  • To make the Ajax examples work, the database file and the directory containing it, must be writable by the webserver user (or chmod a+w examples/data_providers/ examples/data_providers/db.sqlite if you feel brave enough!). You will also need your webserver to load the php sqlite module!
